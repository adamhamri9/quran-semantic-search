# Quran Semantic Search API

Semantic search engine over the Qur'an using embeddings and FAISS.

🚧 **Status:** Work in progress (MVP)

## Goal

Provide an API to search Qur'anic ayat by **meaning**, with optional inclusion of text, tafsir, and translations.

## Tech

* Python
* FAISS
* Sentence Transformers
* FastAPI

## Data Source

* **Qur'an text**: Tanzil Project (used without modification)
* **Tafsir & Translations**: Tarteel QUL project
